# 👑 Devocional Entre Chicas Sabias

Una aplicación web devocional diseñada para mujeres de fe. Bella, sencilla y completamente funcional sin necesidad de backend ni instalación.

## ✨ Características

- 📖 **Devocionales diarios** — 7 devocionales con versículo, reflexión y oración
- ✨ **Versículos bíblicos** — Biblioteca de versículos categorizados con búsqueda y favoritos
- 🙏 **Sección de oración** — Registra peticiones, márcalas como respondidas
- 📝 **Diario espiritual** — Escribe entradas con estado de ánimo y notas personales

## 🚀 Cómo usar

1. Clona o descarga este repositorio
2. Abre `index.html` en tu navegador
3. ¡Listo! No necesitas servidor ni dependencias

```bash
git clone https://github.com/tu-usuario/devocional-entre-chicas-sabias.git
cd devocional-entre-chicas-sabias
# Abre index.html en tu navegador favorito
```

## 🌐 Publicar en GitHub Pages

1. Sube el proyecto a un repositorio en GitHub
2. Ve a **Settings → Pages**
3. En **Source**, selecciona `main` branch y carpeta `/ (root)`
4. ¡Tu app estará disponible en `https://tu-usuario.github.io/devocional-entre-chicas-sabias/`!

## 💾 Almacenamiento

La app guarda tus datos localmente en el navegador (`localStorage`). No se envía ningún dato a servidores externos. Tus oraciones y diario son completamente privados.

## 🛠️ Tecnologías

- HTML5 puro
- CSS3 (variables, animaciones, grid)
- JavaScript vanilla (sin frameworks)
- Google Fonts (Cormorant Garamond + Nunito)
- LocalStorage para persistencia de datos

## 📱 PWA

La app puede instalarse como aplicación en dispositivos móviles usando la opción "Añadir a pantalla de inicio" del navegador.
